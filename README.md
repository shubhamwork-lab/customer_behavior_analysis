# Data Analytics Portfolio Project_1

## Skills & Key Learnings
This project demonstrates practical skills in the following areas:
- **Data Manipulation & Cleaning**: Using Python (pandas) to handle missing values, duplicates, and perform exploratory data analysis (EDA).  
- **SQL & Database Management**: Writing queries in MySQL for aggregations, filtering, and deriving insights.  
- **Data Visualization**: Building interactive dashboards in Power BI to communicate trends and KPIs effectively.
- **End-to-End Analytics Workflow**: Connecting Python, SQL, and visualization tools for a complete data analytics pipeline.

---

## Overview
This project demonstrates the end-to-end process of a data analytics workflow.  
It covers loading and cleaning a dataset, performing EDA, running SQL queries on MySQL, building a Power BI dashboard, creating a report for presenting insights.

The goal is to showcase practical skills in **data analysis, visualization, and reporting**, suitable for professional portfolio display.

---

## Dataset
- The dataset used in this project is about customer shopping behavior.  
- Format: CSV
  
---

## Tools
- **Python**: Pandas  
- **MySQL**: Querying and aggregating data  
- **Power BI**: Building interactive dashboards
- **Jupyter Notebook**: Organizing Python workflows

---

## Steps
1. **Load Data in Python**  
   - Imported CSV dataset using Pandas  
   - Explored structure and summary statistics  

2. **Data Cleaning & EDA**  
   - Handled missing values and duplicates  
   - Visualized distributions, correlations, and trends  

3. **SQL Analysis**  
   - Imported cleaned dataset into MySQL  
   - Ran SQL queries for aggregations, filtering, and insights  

4. **Dashboard Creation**  
   - Connected MySQL queries to Power BI  
   - Built interactive charts, tables, and KPIs  

5. **Report & Presentation**  
   - Compiled findings into a structured report

---

## Dashboard
The Power BI dashboard includes:
- Key metrics and KPIs  
- Interactive charts and tables  
- Filters to explore trends by category, subscription-status, and gender  
- Highlights actionable insights for stakeholders
